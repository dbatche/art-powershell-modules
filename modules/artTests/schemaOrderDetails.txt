Analyzing: POST /orders/{orderId}/details

Operation: OrdersPostDetail
Summary: Add an order detail line
Description: Adds one or more detail line records to a specified order.

The order details contain specific information about the order, such as: warnings, commodity, load dimensions, rating information and more. A single order can have several order detail records.

Order details can be viewed in the Customer Service application > Details tab.
